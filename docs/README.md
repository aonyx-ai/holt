# Holt Documentation

This is the documentation site for Holt, built with
[Docusaurus](https://docusaurus.io/).

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run start

# Or using just
just dev
```

The site will be available at `http://localhost:3000`.

## Build

```bash
npm run build

# Or using just
just build
```

The static output will be in the `build/` directory.

## Deployment

The site is automatically deployed to GitHub Pages when changes are pushed to
`main`.
